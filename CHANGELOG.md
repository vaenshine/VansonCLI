# Changelog

## 1.0.0

- Added injected floating runtime workspace for iOS target apps.
- Added AI Chat with provider selection, context tracking, stop control, references, retryable tool blocks, and verification detail.
- Added model/provider editor with API base URL, API version, endpoint mode, API key, role preset, model list, token limit, reasoning depth, model sync, test call, and call logs.
- Added Objective-C runtime inspector for classes, methods, ivars, properties, protocols, strings, modules, and live instances.
- Added UIKit inspector with touch picking, hierarchy detail, view editing, and optional pick highlight border.
- Added network workspace with request modal tabs, formatted Info/Params/Headers/Body, replay editing, favorites, regex filters, rules, and HAR export.
- Added memory workspace for value scanning, refinement, previews, and controlled write flows.
- Added patch rack for method patches, hooks, value patches, network rules, and retryable patch notes.
- Added artifacts workspace for generated outputs, diagnostics, screenshots, and tool logs.
- Added localization support for Simplified Chinese, Traditional Chinese, English, Arabic, German, Spanish, French, Japanese, Korean, Portuguese, Russian, Thai, and Vietnamese.
- Added VansonCLI branding, About page links, MIT License, and open-source disclaimers.
- Removed legacy access-control UI and related code paths.
